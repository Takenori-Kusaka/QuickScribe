# 貢献者行動規範（Contributor Covenant）

## 私たちの誓い

メンバー、貢献者、およびリーダーである私たちは、年齢、体格、目に見える・見えない障害、民族性、性的特徴、性自認と性表現、経験の水準、教育、社会的・経済的地位、国籍、容姿、人種、宗教、性的指向に関わらず、コミュニティへの参加が誰にとってもハラスメントのない体験となることを誓います。

私たちは、オープンで歓迎的、多様で、包摂的かつ健全なコミュニティに資する形で、行動し交流することを誓います。

## 私たちの基準

私たちのコミュニティにとって望ましい行動の例:

* 他者への共感と優しさを示す
* 異なる意見・視点・経験を尊重する
* 建設的なフィードバックを与え、また快く受け入れる
* 自らの過ちの影響を受けた人々に責任を持って謝罪し、その経験から学ぶ
* 個人だけでなくコミュニティ全体にとって最善であることに焦点を当てる

容認できない行動の例:

* 性的な言葉や視覚表現の使用、あらゆる種類の性的関心や言い寄り
* 荒らし行為、侮辱的・軽蔑的なコメント、個人的・政治的攻撃
* 公的または私的なハラスメント
* 明示的な許可なく、住所やメールアドレスなどの他者の private な情報を公開すること
* 専門的な場で不適切と合理的にみなされるその他の行為

## 施行責任

コミュニティのリーダーは、許容される行動の基準を明確化・施行する責任を負い、不適切・脅迫的・攻撃的・有害とみなした行動に対して、適切かつ公正な是正措置を講じます。

コミュニティのリーダーは、本行動規範に沿わないコメント、コミット、コード、wiki編集、Issue、その他の貢献を削除・編集・拒否する権利と責任を持ち、適切な場合にはモデレーションの理由を説明します。

## 適用範囲

本行動規範は、コミュニティのすべての空間に適用され、また個人が公の場でコミュニティを公式に代表する場合にも適用されます。

## 施行

虐待的・嫌がらせ的・その他容認できない行動は、施行責任者である以下の連絡先に報告できます:

**takenori.kusaka@gmail.com**

すべての苦情は速やかかつ公正にレビューされ、調査されます。コミュニティのリーダーは、いかなる事案の報告者についても、そのプライバシーと安全を尊重する義務を負います。
