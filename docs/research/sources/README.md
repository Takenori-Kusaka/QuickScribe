# Primary Source Snapshots（原典スナップショット）

[ADR-0007](../../adr/0007-research-question-framing-method.md) に基づき、調査が参照した**一次資料を忠実にスナップショット保存**する。

## 方針（著作権配慮）
- 著作権保護された記事/書籍の**全文逐語転載はしない**（Web取得ツールも拒否する）。
- 代わりに各原典について **(1) 正確な書誌・著者・初出 (2) 正典URL (3) 構造（見出し/パターン/手順）(4) 定義など鍵となる短い逐語引用** を保存し、本質を保つ。
- 全文は各URLで参照する。引用は出典明記のうえ最小限にとどめる。
